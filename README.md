# Kasir Sederhana
Bisa digunakan secara langsung ataupun di ubah lagi. Dibuat dengan vue dan electron js.

![Kasir Sederhana](app.png)

## Installasi
- clone atau download repository ini
- `npm insatall`
- `node migration.js`
- `npm run electron:serve`

## Build produksi
- `npm run electron:build`
